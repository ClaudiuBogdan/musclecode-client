# Welcome to MDX

This is a sample MDX file that demonstrates various features.

## Features

- **Component Integration**: You can use React components directly in your markdown
- **Styling**: Custom styling for all markdown elements
- **TypeScript Support**: Full type safety for components and props

## Code Example

```typescript
function greeting(name: string): string {
  return `Hello, ${name}!`;
}
```

## Interactive Components

You can import and use React components directly in your MDX files:

```jsx
import { Button } from "@/components/ui/button";

<Button>Click me!</Button>
```
